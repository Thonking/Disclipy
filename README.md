# Disclipy: A Discord CLI in Python

# Dev

### Setup virtual environment

linux/macos

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

windows

```
python3 -m venv venv
venv\scripts\activate
pip install -r requirements.txt
```

### Push changes

Push all files with:

```
./push.sh
```

Push individual files with:

```
./push.sh file1 file2 file3
```


# Requirements

- Python 3.7
